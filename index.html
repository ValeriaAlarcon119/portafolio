E-Commerce + AI Analytics · 2026

🌿 Mallku Sentires Andinos

From the Andes mountains to your data intelligence center

🌐 Live Store GitHub

Mallku Sentires Andinos is a complete digital ecosystem built for a real Andean medicinal plant brand based in Pasto, Nariño, Colombia. The platform transforms traditional knowledge of botanical medicine — oils, hydrolats, herbs, and ointments — into a modern, premium online shopping experience.

🛒 E-Commerce Store

  • 21 botanical products across 4 categories (Oils, Hydrolats, Herbs, Skincare)
  • Shopping cart with real-time quantity management and price totals
  • Checkout modal: customer data, delivery address, Nequi/Daviplata payment
  • WhatsApp-integrated purchase flow with auto-structured order messages
  • Advisory lead capture form with Netlify Forms + automatic email notifications

📊 AI Analytics Dashboard (React 19)

  • Real-time KPI cards: revenue, conversion rate, active carts
  • Full conversion funnel: Visits → Cart → WhatsApp → Purchase
  • AI Consultant: automatic sales recommendations based on live data
  • Abandoned cart recovery system with re-engagement tracking
  • Customer LTV, product heatmaps, and real-time activity feed

Tech Stack

Store Vanilla JS · HTML5 · CSS3
Backend API Node.js · Express.js
Dashboard React 19 · Vite · Framer Motion
Styling Tailwind CSS · Lucide React
Deploy Netlify · Netlify Forms
Payments Nequi · Daviplata · WhatsApp
